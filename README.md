Blessing Enterprise – Cosmetics Boutique Website

A modern, responsive cosmetics boutique website built using HTML, TailwindCSS, and JavaScript.
The site showcases beauty products, supports a dynamic SPA-style navigation system, and allows customers to place orders directly via WhatsApp.

🚀 Features
SPA-style navigation (pages load dynamically without full refresh)
Fully responsive (mobile → desktop)
Product display system powered by products.js
Add to Cart + WhatsApp Checkout
Floating WhatsApp button
Modern UI using TailwindCSS
Independent pages (Dashboard, Shop, About, Offers, Contact, Location)
